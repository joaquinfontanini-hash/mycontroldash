OBJETIVO DE ESTA ETAPA
Construir el núcleo operativo financiero del sistema y una primera capa real de alertas y vencimientos.

==================================================
1. MÓDULO FINANZAS PERSONALES PRO
==================================================

Crear un módulo de finanzas personales con enfoque ejecutivo.

Debe incluir:

1) Resumen financiero
- patrimonio total
- liquidez disponible
- inversiones
- deudas

2) Tracking manual editable
- caja
- bancos
- cripto
- inversiones
- gastos mensuales
- ingresos si corresponde

3) Métricas automáticas
- variación mensual %
- capacidad de ahorro
- ratio gasto/ingreso
- alerta si gasto supera umbral configurable

4) Alertas financieras
- gasto excesivo
- baja liquidez
- oportunidad detectada
- necesidad de revisión

5) UI
- gráficos simples y claros
- líneas, barras o torta según convenga
- colores:
  - verde = bien
  - amarillo = atención
  - rojo = alerta

==================================================
2. MÓDULO ALERTAS — PRIMERA CAPA REAL
==================================================

Construir un sistema unificado de alertas.

Tipos iniciales:
- financieras
- vencimientos
- recordatorios críticos
- informativas

Debe incluir:
- campanita con contador
- panel desplegable
- clasificación:
  - crítico
  - importante
  - informativo
- lectura/no leída
- timestamp
- persistencia

==================================================
3. MÓDULO VENCIMIENTOS — BASE FUNCIONAL
==================================================

Construir un módulo base de vencimientos.

Debe permitir:
- crear vencimiento manual
- editar

- clasificar

- ver próximos 7 días
- alertar por urgencia

Categorías iniciales:
- impuestos
- cargas sociales
- proveedores
- alquileres
- salud
- otras

Debe existir un widget resumido para el home o la columna lateral futura.
