RESPONSIVE LAYOUT ENGINE
==================================================

Usar dashboard_layouts como fuente de verdad por breakpoint.

Reglas:
- desktop obligatorio
- mobile obligatorio
- tablet opcional

Frontend:
- detectar viewport
- cargar layout correcto
- fallback:
  - si no hay tablet, usar desktop
  - si no hay mobile, apilar automáticamente desde desktop

Builder:
- permitir editar layout desktop y mobile como mínimo
- mostrar claramente qué breakpoint se está editando

No duplicar widgets por breakpoint: cambia el layout, no la identidad del widget.
